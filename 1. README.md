# Operational Issue & Completion Tracking System (Excel)

## Overview
An Excel-based operational tracking system that manages job completion,
inspection stages, and defect closure using formulas, Power Query, and VBA.

## Features
- Advanced Excel formulas (IFS, LET, LOOKUP, TEXTJOIN)
- Power Query for defect consolidation and enrichment
- VBA automation for job and inspection status updates
- Dashboard, KPI summary, and search utilities

## File Structure
- `.xlsm` main workbook
- `/vba` exported VBA modules
- `/screenshots` key visuals

## Notes
- Sample data only
- Business logic generalized for portfolio use
